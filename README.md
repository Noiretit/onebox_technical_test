# Onebox technical test

Aplicación para la empresa Onebox

## Instalación

Usa npm para instalar las dependencias

```bash
npm i
```

## Uso

Se debe arrancar el proyecto y el servidor JSON
```bash
npm start
```
```bash
npm run json:server
```
